# 8 LED Blinking Using AT89C51 Microcontroller

## Description
This project demonstrates 8 LED Blinking using the AT89C51 (8051) microcontroller with Embedded C. All eight LEDs blink simultaneously in a continuous ON/OFF pattern. The project is developed using Keil uVision and simulated in Proteus.

## Components Used
- AT89C51 Microcontroller
- 8 LEDs

## Software Used
- Keil uVision
- Proteus 8
- Embedded C

## Files
- led_blink.c – Embedded C source code
- led_blink.uvproj – Keil project file
- STARTUP.A51 – Startup file
- led_blink.pdsprj – Proteus project file
- Cricuit.cng - circuit diagram
